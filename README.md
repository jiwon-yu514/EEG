# 🧠 EEG

## 연구/프로젝트 개요
뇌전도(Electroencephalography, EEG)는 두피에서 측정한 전기 신호를 통해 **뇌 활동을 실시간으로 관찰할 수 있는 비침습적(non-invasive) 측정 기술**입니다.  
전극을 두피에 부착해 미세한 전위 변화를 증폭·기록함으로써, 윤리적·의학적으로 침습적 시술이 어려운 대상에게도 뇌 활동을 직접 평가할 수 있다는 장점이 있습니다.

하지만 EEG 신호는

- 진폭이 매우 낮고,
- 눈 깜빡임, 근육 움직임, 외부 전기 간섭 등에 쉽게 오염되며,
- 여러 뇌 영역에서 발생한 신호가 두피에서 합쳐진 형태로 관측되기 때문에

**신호대잡음비(Signal-to-Noise Ratio, SNR)가 낮고 공간 해상도가 제한적**이라는 한계를 가집니다.  
또한 피험자 간/세션 간 차이, 장비·환경 변화 등으로 인해 **도메인 간 차이(domain gap)**가 발생하고, 이로 인해 EEG 기반 기계학습 모델은 **일반화 성능이 쉽게 떨어지는 문제**를 갖고 있습니다.

최근에는 이러한 문제를 완화하기 위해

- 피험자 간(inter-subject) 전이 학습,
- 세션 간(inter-session) 전이 학습

등 **Transfer Learning(TL) 기반 프레임워크**들이 활발히 제안되고 있습니다. 그러나 기존 접근법들은 여전히 도메인 갭을 완전히 해소하지 못하고, **계산 비용이 크고 파인튜닝 범위가 넓다**는 한계가 있습니다.

이 레포지토리에서는 이러한 한계를 줄이기 위해,

> **non-CCD(Contrast Change Detection) EEG 데이터로 사전 학습한 인코더를 고정(freeze)한 상태에서,  
> 경량 디코더만 파인튜닝하는 교차-과제(cross-task) EEG 디코딩 모델**

을 탐구합니다.  
즉, **표현(encoder)은 제로샷(zero-shot representation)으로 유지하고, 디코더만 가볍게 적응시키는 방식**으로,  
- 다양한 과제/세션/피험자에도 잘 버티는 **견고한 일반화 성능**과  
- 실용적인 **계산 효율성**을 동시에 달성하는 것을 목표로 합니다.
